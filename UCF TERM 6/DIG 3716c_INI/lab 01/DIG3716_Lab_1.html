Markup the information on this sheet of instructions as an XHTML document according to the criteria listed below:

- Include everything on this sheet in your XHTML document. Yes, that means this line should be there too. And this one! And everything that appears before and after this line. All of the header stuff at the top too. Really. Maybe try using an <h1></h1> element for the “XHTML Lab” text.

Oops – what you should find is that when you tried to write <h1></h1> in the sentence above, the browser didn’t understand that you wanted to literally write those symbols instead of rendering an XHTML element to the screen. A simple way to write the symbols for an XHTML element without rendering the element is to replace the angled brackets <> with their entity character equivalents. The entity character for < is &lt; and the entity character for > is &gt; (the lt stands for less than and the gt stands for greater than, which is what those symbols usually mean when you write them!)

- Don’t worry about the presentation of content – just recreate the structure of this document with valid XHTML
